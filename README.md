# mast387-data-science-lab
# MAST 387 – Data Science Lab Projects

This repository contains selected projects completed for MAST 387 at Concordia University. The work focuses on regression, clustering, dimensionality reduction, and predictive modeling in Python.

## Projects

### Final Project – Air Quality Prediction
This project integrates multiple techniques across numerical methods, linear algebra, and machine learning.

### Part A – Sparse Recovery (MP vs OMP)
Compared Matching Pursuit (MP) and Orthogonal Matching Pursuit (OMP) for signal reconstruction. OMP converges significantly faster and achieves near-zero approximation error.

### Part B – SVD and Image Compression
Applied Singular Value Decomposition (SVD) to analyze image structure. Demonstrated that a small number of singular values capture most of the image information, enabling efficient compression.

### Part C – Housing Data Analysis
Performed exploratory data analysis, dimensionality reduction (PCA), and regression modeling. Results show that housing prices depend on combinations of features rather than a simple low-dimensional structure.

### Part D – Air Quality Prediction
Built Gaussian Process and Random Forest models to predict benzene concentration (`C6H6(GT)`) from sensor data. Both models achieved very high predictive accuracy due to strong feature correlations, with Gaussian Process performing slightly better.

### Key takeaway
This project demonstrates a complete pipeline combining mathematical modeling, data analysis, and machine learning with strong emphasis on interpretation and diagnostics.


### Assignment 3 – Regression and Causal Analysis
Regression modeling and causal inference using logistic regression, weighted regression, and difference-in-differences.

### Assignment 4 – Clustering Analysis
Unsupervised learning on wine chemistry data, with validation against known wine quality labels.

### Midterm – Statistical Learning Exercises
A collection of shorter data science exercises involving visualization, matrix methods, and exploratory analysis.
