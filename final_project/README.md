# Final Project

This project covers multiple core topics in statistical learning, including sparse recovery, dimensionality reduction, and predictive modeling.

## Part A – Sparse Recovery (MP vs OMP)
Compared Matching Pursuit (MP) and Orthogonal Matching Pursuit (OMP) for signal reconstruction. OMP converged significantly faster and achieved near-zero error.

## Part B – SVD and Image Compression
Used Singular Value Decomposition to analyze image structure. Showed that only a small number of singular values are needed to capture most of the image information.

## Part C – Housing Data Analysis
Performed exploratory analysis, PCA, and regression modeling on housing prices. Found that price is influenced by multiple interacting features and cannot be perfectly reduced to low dimensions.

## Part D – Air Quality Prediction
Built machine learning models (Gaussian Process and Random Forest) to predict benzene concentration from sensor data. Both models achieved very high accuracy due to strong correlations among features.

## Key takeaway
This project demonstrates a full workflow from mathematical modeling to machine learning, including interpretation, diagnostics, and model comparison.
